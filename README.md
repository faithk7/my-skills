# My Skills

Snapshot of the local Codex skill directories copied into this repository.

## Repository Info

- Git remote: `git@github.com:faithk7/my-skills.git`
- GitHub URL: https://github.com/faithk7/my-skills
- Captured from: `~/.codex/skills` and `~/.agents/skills`
- Skill count: 29
- Machine-readable manifest: [`skills-manifest.json`](skills-manifest.json)

## Restore On Another Machine

Clone the repo, then sync the Codex skills and the agent skills back to their original locations:

```bash
git clone git@github.com:faithk7/my-skills.git ~/Developer/my-skills
repo=~/Developer/my-skills
mkdir -p ~/.codex/skills ~/.agents/skills

rsync -a \
  --exclude '.git/' \
  --exclude 'README.md' \
  --exclude 'skills-manifest.json' \
  --exclude 'microsoft-foundry/' \
  "$repo"/ ~/.codex/skills/

rsync -a "$repo"/microsoft-foundry/ ~/.agents/skills/microsoft-foundry/
```

## Layout

- Everything in this repo except `microsoft-foundry/`, `README.md`, and `skills-manifest.json` belongs under `~/.codex/skills/`.
- `microsoft-foundry/` belongs under `~/.agents/skills/microsoft-foundry/`.
- Hidden system skills are stored in `.system/` and should remain nested under `~/.codex/skills/.system/`.
